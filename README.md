# 📊 Sales Analysis and Prediction System

## 📌 Overview
This project focuses on analyzing sales data and building insights using data visualization and machine learning techniques. It helps understand sales patterns across categories, regions, segments, and top-selling products.

---

## 📁 Project Structure
- `script.ipynb` → Main Jupyter notebook for analysis  
- `train.csv` → Dataset used for training and analysis  
- `salespulse_clean.csv` → Cleaned dataset  
- `.png files` → Visualizations and charts  

---
kaggel dataset link- https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting
---

## 📊 Key Analysis Performed
- Sales trend analysis over time  
- Category-wise sales distribution  
- Region-wise sales performance  
- Segment-wise sales comparison  
- Top 10 products analysis  
- Correlation heatmap for feature relationships

## 📈 Visualizations Included
- Heatmap of correlations

- Sales by category

- Top 10 products chart
